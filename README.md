# SENG550 - Large Scale Data Analysis

This repository contains assignment files for SENG550 (Large Scale Data Analysis).

## Course Description

Sources and characteristics of large scale data, i.e., "big data", large scale data analysis, benefits of large-scale data analysis to various industry domains, programming paradigms and middleware technologies for scalable data analysis, algorithms that enable large-scale data processing and application of large scale data algorithms.

## Assignments

This repository contains the following assignments:

- **Assignment1**: Database operations and SQL queries
- **Assignment2**: ETL processes and data warehousing (PostgreSQL and MongoDB)
- **Assignment3**: Large-scale data processing with Airflow and machine learning pipelines

Each assignment directory contains its own README with specific instructions and details.
